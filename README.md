# Smart Bookmark App

A real-time bookmark manager where users sign in with Google and manage private bookmarks.

## Live App

https://smart-bookmarks-3ymywkb2h-bhaurao8530s-projects.vercel.app/

## Tech Stack

* Next.js (App Router)
* Supabase Auth & Database
* Supabase Realtime
* Tailwind CSS
* Vercel Deployment

## Features

* Google OAuth login
* Add bookmarks (title + URL)
* Bookmarks are private per user
* Real-time updates across tabs
* Delete bookmarks
* Deployed on Vercel

## Run Locally

Clone repo

```
git clone <your-repo-url>
cd smart-bookmarks
```

Install packages

```
npm install
```

Create `.env.local`

```
NEXT_PUBLIC_SUPABASE_URL=your_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_key
```

Start project

```
npm run dev
```
